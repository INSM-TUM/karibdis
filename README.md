# knowledge-graph-resource-allocation

## Prerequisites
This project uses Python 3.

First, install the necessary Python libraries:
```console
pip install -r requirements.txt
```

This repository additionally utilizes the BPO project simulation frame. Please download it from the original source (we used revision [f1f90a3](https://github.com/bpogroup/bpo-project/commit/f1f90a320812cb21606c79bbb1e96b056266ba26)) and install the respective required packages:
```console
cd ./knowledge-graph-resource-allocation
git clone git@github.com:bpogroup/bpo-project.git
pip install -r ./bpo-project/requirements.txt
```
